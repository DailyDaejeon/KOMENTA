# README



## scss 폴더 구조 설명

- abstracts : 변수, mixin 함수

- base : 공통적으로 쓰이는 기본 

- layout : 개별 파일에서 레이아웃(헤더, 푸터 등)의 주요 부분에 관한 스타일을 정의 

- components: 레이아웃보다 작은 요소를 넣음(로그인, 타이틀)
